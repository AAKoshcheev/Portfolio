| Название проекта | Описание | Используемые библиотеки |

| [Исследование технологического процесса очистки золота](big_cities_music) | Строитстся модель машинного обучения для промышленной компании, разрабатывающая решения для эффективной работы промышленных предприятий. Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды на основе данных с параметрами добычи и очистки. Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками.| * Python, Pandas, Matplotlib, NumPy, Scikit-Learn* |
